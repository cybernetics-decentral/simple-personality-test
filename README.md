# simple-personality-test
A simple MBTI personality quiz built as a Progressive Web App.

## Usage

1. Open `index.html` in a browser or serve the folder locally.
2. Answer the four questions.
3. View your MBTI profile page, including childhood, family, friends, career, relationships, parenthood, and growth path.

## Run locally

```bash
cd /workspaces/simple-personality-test
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.
